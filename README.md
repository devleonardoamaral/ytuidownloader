# ytuidownloader - Youtube Video Downloader

This project was made purely for the purpose of study and curiosity.

## How to execute
```bash
python -m venv .venv
```
```bash
# Linux
source .venv/bin/activate

# Windows
.\.venv\Scripts\activate
```
```bash
pip install -r requirements.txt
```
```bash
python src/ytuidownloader/main.py
```
